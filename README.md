# metalooze.vercel.app

## Stack
- Next.js
- tRPC
- Prisma
- NextAuth

## Running Locally
This application requires Node.js to run

```
git clone https://github.com/MetalOoze05/website.git
cd website
npm i -g pnpm
pnpm i 
pnpm dev
```
